#### Getting started

**with Makefile**
```shell
make build
make run
```
**on command line**
```shell
python -m venv .venv
. .venv/bin/activate # linux/macos 
# venv/Scripts/activate on Windows
pip install -r requirements.txt

python run.py
```

open [here](http://localhost:8000)!
